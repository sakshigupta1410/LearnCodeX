# **Introduction to HTML**

**HTML (HyperText Markup Language)** is the standard markup language used to create **web pages**.  
It describes the structure of a webpage using **elements** and **tags**.

---

## **Key Features of HTML**

- **Structure**: Defines the structure of content on a webpage.
- **Markup**: Uses tags to mark up text, images, links, and other content.
- **Hypertext**: Allows linking between pages using hyperlinks.
- **Platform Independent**: Can be viewed in any web browser.

---

## **HTML Document Structure**

A basic HTML document has the following structure:

```html
<!DOCTYPE html>
<html>
<head>
    <title>My First HTML Page</title>
</head>
<body>
    <h1>Welcome to HTML!</h1>
    <p>This is my first HTML page.</p>
</body>
</html>
```

<br>

**Explanation**:

* html → Declares the document type (HTML5).

* html → Root element of the HTML document.

* head → Contains metadata like title, scripts, and styles.

* title → Title of the webpage (appears in the browser tab).

* body → Contains the content displayed on the webpage.

* h1 → Heading element (h1 to h6 for levels).

* p → Paragraph element.

## **HTML Tags**

Tags are enclosed in angle brackets **< >**.

Most tags have an opening tag and a closing tag.

**Example**: 

```
<p>Paragraph</p>
```
<br>

## **Common HTML Elements**

| **Element**     | **Description**                    |
|-----------------|------------------------------------|
| `<h1>` to `<h6>` | Headings (1 is largest)            |
| `<p>`           | Paragraph                          |
| `<a>`           | Anchor (link)                      |
| `<img>`         | Image                              |
| `<ul>` / `<ol>` | Unordered / Ordered List           |
| `<li>`          | List Item                          |
| `<div>`         | Division or container              |
| `<span>`        | Inline container                   |


## **Summary**

* HTML is the **_backbone_** of web pages.
 
* Uses tags to define elements.
 
* HTML documents have a structured layout with `<html>`, `<head>`, and `<body>`.
 
* Can include headings, paragraphs, links, images, and lists.
