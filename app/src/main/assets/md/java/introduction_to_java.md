# **🔹 What is Java?**

Java is a **high-level**, **class-based**, **object-oriented programming language** designed to have as few implementation dependencies as possible.

- Developed by **Sun Microsystems** in 1995 (now owned by Oracle).
- Famous for the slogan: **"Write Once, Run Anywhere"** (WORA).
- Java programs run on the **Java Virtual Machine (JVM)**, making them platform-independent.

---

## **🔹 Key Features of Java**

* Object-Oriented
* Platform Independent
* Simple
* Secure
* Robust
* Multithreaded
* High Performance
* Distributed
* Dynamic
---

## **🔹 How Java Works?**

1. **Write** source code (`.java` file).
2. **Compile** source code into bytecode (`.class` file) using `javac`.
3. **Run** bytecode on JVM.

Source Code (Java) → Compiler (javac) → Bytecode (.class) → JVM → Machine Code (Executed)

---

## **🔹 Java Program Structure**
<br>

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, LearnCodeX!");
    }
}
```

**public class HelloWorld** — _Defines a class._

**public static void main(String[] args)** — _Main method, entry point of any Java program._

**System.out.println()** — _Prints text to the console._

## **🔹 JVM, JRE, JDK**

**JVM**	- Java Virtual Machine: 
-Runs Java bytecode

**JRE**	- Java Runtime Environment:
-JVM + standard libraries

**JDK**	- Java Development Kit: 
-JRE + development tools (compiler, debugger, etc.)

## **🔹 Java Development Workflow**

**Writing Code** - Any text editor or IDE (Eclipse, IntelliJ)

**Compiling** - `javac` compiler

**Running** - `java` command to launch JVM

**Debugging** - Tools like `jdb`, IDE built-in debuggers

## **🔹 Summary**

- Java is a portable, secure, and powerful programming language.
- Platform-independent via the JVM.
- Uses an object-oriented approach.
- Supports multithreading and networking.
- Java programs go through compile and run phases.
