# Loops in Java

Loops are used to execute a block of code repeatedly.

**for loop:**
```java
for(int i = 0; i < 5; i++) {
    System.out.println(i);
}
```

**while loop:**
```java
int i = 0;
while(i < 5) {
    System.out.println(i);
    i++;
}
```

**do-while loop:**
```java
int i = 0;
do {
    System.out.println(i);
    i++;
} while(i < 5);
```
