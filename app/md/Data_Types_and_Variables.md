# Data Types and Variables

<span style="color:#3F51B5;">Java supports several data types, which are classified into two categories: primitive and reference.
</span>
**Primitive Data Types:**
- byte, short, int, long
- float, double
- char
- boolean

**Example:**
```java
int number = 15;
double price = 99.99;
char grade = 'A';
boolean isJavaFun = true;
```
