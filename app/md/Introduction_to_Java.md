<h1 style="color:#3F51B5;">Java Programming Language</h1>

<p style="color:#666;">Java is a <b style="color:#4CAF50;">high-level</b>, <b style="color:#FF5722;">object-oriented</b>, and <b style="color:#03A9F4;">platform-independent</b> programming language.</p>

---

<h2 style="color:#673AB7;">✨ Features of Java</h2>

- <span style="color:#009688;">Simple</span>
- <span style="color:#E91E63;">Object-Oriented</span>
- <span style="color:#00BCD4;">Portable</span>
- <span style="color:#FFC107;">Platform Independent</span>
- <span style="color:#9C27B0;">Secure</span>
- <span style="color:#4CAF50;">Robust</span>

---

<h2 style="color:#FF9800;">📌 Java Code Example</h2>

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}
```
